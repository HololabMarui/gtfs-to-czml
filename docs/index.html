GTFS → CZML / GeoJSON 補助ツール

GitHub Pages で公開する前提の、明るめUIのモックアップです。ツール側を主画面にしつつ、チュートリアルとPython配布を別タブに分けています。 CZML出力系とGeoJSON出力系もレイアウトを分離し、GeoJSON側ではアイコン集から marker-symbol を選べる構成にしています。

静的サイト / ローカル実行補助 / GitHub Pages向け

CZML 出力ツール

3Dモデル走行
GTFS-JP から CZML を生成するための入力群です。現在の Python は --gtfs-dir を前提にしているため、ZIPのまま直接渡すのではなく、 このUIでは ZIPを指定した場合に展開コマンドも一緒に生成する 形にしています。

基本設定

GTFS入力、日付、出力ファイル、シェル形式などを指定します。

Windowsなら python や py に変更できます。
ZIPモード時に展開先として使います。

アニメーション・描画オプション

shape がない便の扱い、モデル描画、補間などを指定します。

GeoJSON 出力ツール

停留所 / marker属性
stops.txt から GeoJSON を出力するコマンドと、必要なら marker 属性付与コマンドを別系統で生成します。 marker-symbol はアイコン集から選べるようにしています。

停留所 GeoJSON 出力

GTFSから停留所ポイントをGeoJSON化します。route_id 絞り込みも可能です。

marker アイコン選択

simplestyles のアイコン集 GeoJSON を読み込み、marker-symbol 候補として選べるようにしています。

選択中: なし
選択したアイコン名は --marker-symbol としてコマンドに反映します。

追加の marker 属性付与

GeoJSONに対して後から marker 属性を付与したい場合の補助コマンドです。

環境設定・チュートリアル

        

        

          

            

              
基本の流れ

              

                1. Python をインストール

                2. このページから Python スクリプトをダウンロード

                3. GTFS-JP を ZIP のまま置くか、手動で展開

                4. ツールタブでオプションを選択

                5. 生成されたコマンドをローカル端末で実行
              

            

            

              
ZIP入力について

              

                現行スクリプトは --gtfs-dir 前提です。そのため、ZIPモードではこのページが先に展開コマンドを出し、その後に Python コマンドを続ける想定です。
                将来的には Python 側を改修して ZIP を直接読めるようにもできます。
              

            

          


          

            
Mac / Linux の例

            
unzip -o "./GTFS_JP_FEED.zip" -d "./GTFS_JP_FEED"
python3 gtfsjp_to_czml.py \
  --gtfs-dir "./GTFS_JP_FEED" \
  --service-date "2026-03-12" \
  --output "bus_animation.czml" \
  --model-url "https://example.com/models/bus.glb"

          


          

            
Windows PowerShell の例

            
Expand-Archive -Path "./GTFS_JP_FEED.zip" -DestinationPath "./GTFS_JP_FEED" -Force
python gtfsjp_to_czml.py `
  --gtfs-dir "./GTFS_JP_FEED" `
  --service-date "2026-03-12" `
  --output "bus_animation.czml" `
  --model-url "https://example.com/models/bus.glb"
